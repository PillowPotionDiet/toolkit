# Image & Video Compressor Tool

## Folder Structure
Place your compressor tool files here:

- `index.html` - Main tool page (handles both image and video compression)
- `css/` - Stylesheets
- `js/` - JavaScript files
- `assets/` - Images and other assets

## Features
This tool handles:
- Image compression (JPG, PNG, WebP)
- Video compression (MP4, MOV, AVI)
- Image format conversion

## Access URL
Once deployed: https://toolkit.pillowpotion.com/compressor/

## Note
Copy your existing Compressor code from the separate project into this folder.
